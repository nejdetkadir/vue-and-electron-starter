## Project setup
```
npm install
```

### Start development server
If you use NPM:
```
npm run electron:serve
```
If you use Yarn:
```
yarn electron:serve
```

### Build application
With NPM:
```
npm run electron:build
```
or with Yarn:
```
yarn electron:build
```
